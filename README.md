## Config
<details>
<summary>Redie.json</summary>
  
```json
{
  "Prefix": "{purple}[Redie]{grey}",
  "Commands": ["css_redie", "css_ghost"],
  "Messages": true,
  "Message_Redie": "You have been respawned as a ghost",
  "Message_UnRedie": "You are no longer a ghost"
}
```
